# Bellavella Documentation

This documentation set covers the Bellavella Flutter mobile app and the Laravel backend used by the project.

## Structure

- `project-overview.md` — high-level architecture, project scope, and workspace layout
- `flutter-architecture.md` — Flutter app architecture, routing, state, API integration, and platform setup
- `backend-architecture.md` — Laravel backend architecture, route groups, auth, wallets, and kit store workflows
- `api-reference.md` — concise API endpoint inventory for client, professional, and admin workflows
- `database-wallet-models.md` — wallet data model, database schema, and wallet semantics for professionals and clients

## Usage

Open the files above to understand the project structure, core flows, and implementation details.

> The backend code is located in the sibling workspace `c:\xampp\htdocs\bellavellaa`.
> Existing backend API documentation is also available under `bellavellaa/docs/api`.
